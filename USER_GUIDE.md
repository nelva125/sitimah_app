# Panduan Penggunaan Aplikasi siTIMAH

## 1. Persiapan & Instalasi
Aplikasi ini sudah siap dijalankan (Portable).
- **Android**: Install file APK `sitimah.apk` di HP Android Anda.
- **Windows**: Jalankan `sitimah.exe` di folder Release (bila tersedia).
- **Development**:
  1. Pastikan Flutter SDK terinstall.
  2. Buka terminal di folder project.
  3. Ketik `flutter run` untuk menjalankan.

## 2. Cara Masuk (Login)
- Buka aplikasi siTIMAH.
- Halaman Login akan muncul.
- Gunakan akun demo berikut:
    - **Username**: `admin`
    - **Password**: `admin123`
- Klik tombol **Masuk**.

## 3. Mencatat Pembelian Timah (Fitur Utama 1)
1. Setelah login, Anda akan langsung berada di menu **Pencatatan**.
2. **Tanggal**: Otomatis terisi hari ini, klik ikon kalender jika ingin mengubah.
3. **Pemasok**: Ketik nama penambang/pemasok (Misal: Pak Budi).
4. **Jumlah (Kg)**: Masukkan berat bersih timah (Misal: 12.5).
5. **Harga/Kg**: Masukkan harga beli per kilo (Misal: 125000).
6. **Kualitas**: (Opsional) Masukkan grade timah (A/B).
7. **Simpan**: Klik tombol **Simpan Pembelian**.
   - Muncul notifikasi "Data berhasil disimpan!" jika sukses.

## 4. Melihat Rekap Mingguan (Fitur Utama 2)
1. Klik menu **Rekap** (ikon Kalender) di bagian bawah.
2. Anda akan melihat Dashboard yang berisi:
    - Total Transaksi minggu ini.
    - Total Berat (Ton/Kg) terkumpul.
    - Total Uang yang dikeluarkan.
    - Grafik sederhana tren harga/berat.
3. Ini membantu Anda mengetahui *cashflow* hari ini.

## 5. Melihat & Menghapus Riwayat
1. Klik menu **Riwayat** (ikon Jam).
2. Daftar transaksi akan muncul dari yang terbaru.
3. Gunakan kolom **Pencarian** di atas untuk mencari nama pemasok.
4. Klik ikon **Sampah (Merah)** pada kartu transaksi untuk menghapus data yang salah.

## 6. Keluar
- Klik tombol **Keluar** di pojok kanan atas layar mana saja, atau melalui menu **Profil**.
